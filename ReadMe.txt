功能特点：
1. 不依赖于重现性，亦即不依赖于工况稳定性
2. 既可计算加料量、也可计算加料时点，不依赖于预设模式
3. 根据实际工况确定供氧制度，不依赖于预设模式
4. 提供全程熔池状态趋势，含钢液及炉渣的成分、温度和重量
5. 既可用于BOF炉，也可以用于AOD炉
使用要点：
1. 提供核心算法及其演示
2. 外部数据通过//应用文件/Data.txt读入，运行程序前请确保该文件准确、完整
3. 计算结果通过：1）//应用文件/Test.txt输出，可使用EXCEL文件打开（间隔符）；2）屏幕输出少量结果
应用构成：
1. 常用功能模板
2. 数据元操作
3. 数据元向量操作
4. 外部数据获取及初步处理
5. 策略规划：1）熔剂；2）合金；3）能量；4）氧量；5）加料方案；6）供氧方案
6. 熔池状态
7. 吹炼中修正
8. 合金化/还原（不锈钢）
